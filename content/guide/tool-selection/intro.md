에이전틱 코딩 도구는 크게 3가지입니다. **안티그래비티**, **클로드코드**, **오 마이 클로드코드(OMC)**. 어떤 걸 선택해야 할까요?

<Callout type="success" title="🎯 30초 결론">
- **완전 초보**: 안티그래비티 (GUI, 시각적)
- **약간 아는 사람**: 클로드코드 (터미널, 정확)
- **고급 사용자**: OMC (클로드코드 + 확장팩)
</Callout>

## 🎨 1. 안티그래비티 (Google Antigravity)

### 이런 분께 추천

<Callout type="info" title="✅ 안티그래비티를 선택하세요">
- 코딩을 한 번도 안 해봤다
- 터미널(검은 화면)이 무섭다
- 마우스로 클릭하는 게 편하다
- 이미지 생성도 같이 하고 싶다
- 브라우저 자동화가 필요하다
</Callout>

### 장점

**1. 사용하기 쉬움**
- GUI(그래픽 인터페이스) 제공
- 마우스 클릭만으로 모든 작업 가능
- 시각적 피드백 즉시 확인

**2. 통합 기능**
- **Nanobana**: 이미지 생성 (DALL-E 스타일)
- **Browser Agent**: 웹사이트 자동 테스트
- **Agent Manager**: 여러 AI 에이전트 동시 실행

**3. 초보자 친화적**
- 에러 메시지가 친절함
- 자동 수정 제안
- 튜토리얼 내장

### 단점

**1. 정확도가 낮음**
- Gemini 모델의 한계
- 복잡한 로직 처리 시 실수 많음

**2. 속도가 느림**
- GUI 렌더링 오버헤드
- 브라우저 자동화 시 대기 시간

**3. 커스터마이징 제한**
- 설정 옵션이 적음
- 고급 기능 부족

### 시작하기

```
1. 공식 사이트에서 설치 파일 다운로드
2. 더블 클릭으로 설치
3. GUI 실행
4. "New Project" 클릭
5. 채팅창에 요청 입력
```

**공식 사이트**: (Google Antigravity 검색)

## 💻 2. 클로드코드 (Claude Code)

### 이런 분께 추천

<Callout type="info" title="✅ 클로드코드를 선택하세요">
- 터미널을 좀 써봤다
- 정확한 결과가 중요하다
- 복잡한 프로젝트를 만들고 싶다
- 빠른 작업 속도를 원한다
- 프로 개발자처럼 일하고 싶다
</Callout>

### 장점

**1. 최고 수준의 정확도**
- Claude Opus 4.6 모델
- 논리적 추론 능력 최상위
- 복잡한 코드도 정확히 이해

**2. 빠른 속도**
- 터미널 기반 (GUI 없음)
- 키보드만으로 모든 작업
- 불필요한 렌더링 없음

**3. 강력한 기능**
- Plan Mode: 실행 전 계획 수립
- 서브에이전트: 역할별 AI 분리
- 훅(Hooks): 자동화 설정
- 스킬(Skills): 커스텀 명령어

**4. 생태계**
- 9,000+ 플러그인
- 활발한 커뮤니티 (Discord 59k+)
- 풍부한 튜토리얼

### 단점

**1. 학습 곡선**
- 터미널 명령어 익혀야 함
- 초기 설정 복잡
- 개념 이해 필요 (Plan Mode, 서브에이전트 등)

**2. 비용**
- 월 $20 구독
- API 토큰 비용 별도

**3. GUI 없음**
- 시각적 피드백 부족
- 결과 확인을 위해 별도 브라우저 필요

### 시작하기

**macOS:**
```bash
# Homebrew 설치 (터미널 실행)
brew install claude-code

# 실행
claude code

# 또는 GUI 설치 파일 다운로드
ClaudeCodeInstaller.dmg
```

**Windows:**
```powershell
# winget으로 설치 (PowerShell)
winget install AnthropicPBC.ClaudeCode

# 또는 GUI 설치 파일 다운로드
ClaudeCodeInstaller.exe
```

**공식 문서**: https://code.claude.com/docs

## 🚀 3. 오 마이 클로드코드 (OMC)

### 이런 분께 추천

<Callout type="info" title="✅ OMC를 선택하세요">
- 클로드코드를 이미 쓰고 있다
- 더 강력한 기능이 필요하다
- 멀티 에이전트 오케스트레이션을 하고 싶다
- 자동화를 극대화하고 싶다
- 커뮤니티 플러그인을 적극 활용하고 싶다
</Callout>

### 장점

**1. 클로드코드의 모든 기능 + α**
- 기본 기능 그대로
- 추가 확장팩 제공

**2. 고급 에이전트 시스템**
- **Autopilot**: 아이디어 → 완성 자동화
- **Ralph**: 자율 루프 (완료까지 반복)
- **Ultrawork**: 최대 병렬 처리
- **Team**: N개 에이전트 협업

**3. 무료**
- 플러그인 형태라 추가 비용 없음
- 클로드코드 구독만 있으면 OK

**4. 커뮤니티 기반**
- 오픈소스
- 활발한 개발
- 커스터마이징 자유

### 단점

**1. 복잡도 높음**
- 개념 이해 필요 (Autopilot, Ralph, Ultrawork 등)
- 설정 파일 작성 필요
- 디버깅 어려움

**2. 불안정성**
- 공식 지원 아님
- 버그 가능성
- 문서 부족

**3. 클로드코드 필수**
- OMC 단독 사용 불가
- 클로드코드 먼저 익혀야 함

### 시작하기

```bash
# 1. 클로드코드 설치 (필수)
# (위의 클로드코드 설치 방법 참고)

# 2. OMC 플러그인 설치
claude plugin add oh-my-claudecode

# 3. 셋업
/oh-my-claudecode:omc-setup

# 4. Autopilot 실행 예시
/oh-my-claudecode:autopilot "블로그 만들어줘"
```

**공식 저장소**: (GitHub에서 'oh-my-claudecode' 검색)
