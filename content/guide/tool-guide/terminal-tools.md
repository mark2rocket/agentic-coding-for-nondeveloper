## 터미널 도구 소개

터미널은 개발자의 필수 도구입니다. 에이전틱 코딩을 더 효율적으로 하려면 좋은 터미널이 필요합니다.

### iTerm2

<Callout type="info" title="💻 macOS 전용">
iTerm2는 macOS를 위한 가장 인기 있는 터미널 대체 앱입니다.
</Callout>

**핵심 기능**:
- 분할 패널 (수평/수직)
- AI 지원 (2025년 추가)
- 무제한 명령 히스토리 (최대 4M)
- tmux 통합
- 커스터마이징 가능한 테마

**장점**: 완전 무료 오픈소스, macOS 네이티브 UX, 풍부한 기능

**단점**: macOS 전용, 가끔 창 크기 조절 이슈

---

### Warp

<Callout type="tip" title="🤖 AI 네이티브">
Warp는 AI가 내장된 차세대 터미널입니다.
</Callout>

**핵심 기능**:
- AI 명령어 제안 (자연어로 명령어 설명 → 실행)
- 블록 기반 명령어 (명령어별 독립적 블록)
- 여러 AI 모델 지원 (Claude, GPT, Gemini)
- 협업 기능 (워크플로우 공유)
- 데이터 학습 차단 (프라이버시 보호)

**장점**: AI 통합, 빠른 렌더링, 팀 협업, 최신 모델 즉시 사용

**단점**: 무료 티어 제한, 상대적으로 새로운 도구

---

### Kitty

<Callout type="success" title="⚡ 최고 성능">
Kitty는 GPU 가속으로 최고의 성능을 자랑합니다.
</Callout>

**핵심 기능**:
- GPU 가속 (OpenGL)
- 초고속 텍스트 렌더링 (1500+ chars/s)
- 탭/타일링 레이아웃
- 강력한 스크립팅 (kittens)
- Kitty Image Protocol

**장점**: 최고 성능, 25% 낮은 CPU 사용, 크로스플랫폼 (Linux/macOS/BSD)

**단점**: 설정이 복잡할 수 있음

---

### Ghostty

<Callout type="info" title="🆕 신예 강자">
HashiCorp 공동창업자 Mitchell Hashimoto가 만든 최신 터미널입니다.
</Callout>

**핵심 기능**:
- GPU 가속
- 네이티브 탭/분할
- Kitty 그래픽 프로토콜 지원
- 라이트/다크 모드 자동 전환
- 조용한 업데이트 알림

**장점**: 빠른 성능, 모던한 기능, 크로스플랫폼

**단점**: 상대적으로 새로운 도구 (생태계 성장 중)

---

### tmux

<Callout type="warning" title="🔧 멀티플렉서">
tmux는 터미널 에뮬레이터가 아닌 터미널 멀티플렉서입니다.
</Callout>

**핵심 기능**:
- 세션 관리 (연결 끊어도 프로세스 유지)
- 분할 패널 (여러 셸을 하나의 창에서)
- SSH 연결 유지
- Vi 키바인딩
- 가벼운 메모리 사용

**장점**: 가볍고 안정적, 서버에서 필수, 어디서나 실행 가능

**단점**: 초보자에게 복잡한 키바인딩, 마우스 지원 제한적

**사용 팁**: 다른 터미널 에뮬레이터(iTerm2, Kitty 등)와 함께 사용하면 시너지 효과!

## 선택 가이드

**완전 초보자** → **iTerm2** (macOS) 또는 **Warp** (모든 플랫폼)
- 설치 쉽고, GUI 직관적
- AI 기능으로 명령어 배우기 좋음

**중급 개발자** → **Warp** 또는 **Ghostty**
- AI로 생산성 극대화
- 모던한 개발 워크플로우

**성능 중시** → **Kitty**
- GPU 가속으로 최고 속도
- 대량 로그 처리에 탁월

**서버 작업 많음** → **tmux** (필수)
- SSH 연결 끊어도 세션 유지
- 원격 서버에서 필수 도구
- 다른 터미널과 함께 사용 추천

**조합 추천**:
- **로컬**: Warp (AI + 빠른 렌더링)
- **원격**: tmux (세션 관리)
- **조합**: Warp + tmux = 최강의 조합!
