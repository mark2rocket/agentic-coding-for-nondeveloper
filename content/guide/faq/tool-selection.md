## Q6: 웹 IDE vs 클로드코드, 뭘 써야 하나요?

**선택 플로우**:

```
Q: 완전 처음이신가요?
├─ Yes → 웹 IDE (StackBlitz 추천)
│   이유: 설치 불필요, 시각적
│
└─ No → Q: 복잡한 로직 다룰 예정?
         ├─ Yes → 클로드코드
         │   이유: 더 강력, 정확
         │
         └─ No → 웹 IDE
             이유: 빠르고 편함
```

**비교**:

| 기준 | 웹 IDE | 클로드코드 |
|------|--------|-----------|
| 시작 속도 | 즉시 | 설치 필요 (10분) |
| 학습 곡선 | 완만 | 약간 가파름 |
| 성능 | 보통 | 우수 |
| 비용 | 제한적 무료 | 사용량 과금 |
| 오프라인 | ❌ | ✅ (일부) |
