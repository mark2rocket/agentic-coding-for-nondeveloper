**설명**: AI 에이전트(클로드코드, 안티그래비티 등)가 사용자의 지시를 받아 실제 프로그래밍 언어로 코드를 작성하고 파일을 생성하는 단계입니다. 사용자는 키보드를 치는 것이 아니라 감독관이 됩니다.

**예시**:

```bash
(AI에게 명령) "방금 기획한 기분 기반 음악 추천 로직을 Next.js로 짜고,
필요한 컴포넌트 파일을 생성해. 스타일은 Tailwind CSS를 써서 적용해줘."
```

**부연 설명**: 이 단계에서 AI는 순식간에 수십 개의 코드 파일을 생성하고, 필요한 라이브러리를 설치하며, 폴더 구조를 정리합니다. 사용자는 AI가 생성하는 파일명이나 폴더 구조를 보면서 내가 의도한 대로 만들어지고 있는지 확인만 하면 됩니다. AI가 멈추거나 모호한 점을 되물어볼 수 있는데, 당황하지 말고 상사가 직원에게 업무 지시를 내리듯 방향을 정해주면 됩니다. (예: "그건 A 방식으로 해")

**한줄 요약**: AI는 당신의 손과 발입니다. 팔짱 끼고 모니터를 보며 AI가 일하는 것을 감독하세요.

---

## 🎯 클로드코드로 5분 실습하기

<Callout type="success" title="✅ 실습 목표">
AI가 실제로 파일을 생성하고, 코드를 작성하고, 라이브러리를 설치하는 전 과정을 관찰합니다.
</Callout>

### Step 1: 프로젝트 폴더 생성

```bash
# 실습용 폴더 생성
mkdir ~/agentic-practice
cd ~/agentic-practice

# 클로드코드 실행
claude code
```

---

### Step 2: 간단한 웹사이트 생성 요청

채팅창에 입력:

```
간단한 자기소개 웹사이트를 만들어줘.

내용:
- 제목: "안녕하세요, [이름]입니다"
- 부제: "에이전틱 코딩을 배우는 중입니다"
- 소개 문구 3줄
- 연락 버튼 (클릭하면 이메일 주소 표시)

디자인:
- 배경: 그라데이션 (하늘색 → 보라색)
- 중앙 정렬
- 카드 형태
- 부드러운 애니메이션 (페이드 인)

파일:
- index.html
- style.css
- script.js
```

---

### Step 3: AI 작업 과정 관찰하기

AI가 다음 작업을 자동으로 수행하는 것을 확인하세요:

**1단계: 파일 생성**
```
✓ Writing index.html...
✓ Writing style.css...
✓ Writing script.js...
```

**2단계: 코드 작성**
- HTML 구조 생성
- CSS 스타일 적용
- JavaScript 이벤트 추가

**3단계: 파일 연결**
- HTML에서 CSS, JS 링크
- 모든 파일이 올바르게 연결됨

**총 소요 시간**: 약 30초

---

### Step 4: 결과 확인하기

**방법 1: 브라우저에서 직접 열기**
```bash
# Mac
open index.html

# Windows
start index.html
```

**방법 2: 로컬 서버 실행 (권장)**
```bash
# Python 설치되어 있으면
python -m http.server 8000

# 또는 npx 사용
npx serve
```

브라우저에서 `http://localhost:8000` 접속

---

### Step 5: 생성된 파일 구조 확인

```bash
ls -la
```

**출력 예시:**
```
index.html      # HTML 구조
style.css       # 디자인 스타일
script.js       # 인터랙션 로직
```

---

### Step 6: 코드 내용 확인 (선택)

AI가 작성한 코드를 직접 열어볼 수 있습니다:

```bash
cat index.html
```

**확인할 점:**
- 요청한 내용이 모두 포함되었는가?
- 파일 간 연결이 올바른가?
- 코드가 깔끔하고 정리되어 있는가?

---

### 💡 실습에서 배운 점

<Callout type="tip" title="Generation 단계의 핵심">
1. **AI는 놀라울 정도로 빠릅니다** - 30초 만에 3개 파일 완성
2. **사람보다 실수가 적습니다** - 오타, 연결 누락이 거의 없음
3. **파일 구조를 자동으로 정리합니다** - 폴더 구조 고민 불필요
4. **감독만 하면 됩니다** - 코드를 직접 볼 필요도 없음
</Callout>

**주의사항:**
- AI가 작업 중에 질문하면 명확하게 답변하세요
- 결과가 마음에 안 들면 다음 단계(Iteration)에서 수정합니다

**소요 시간**: 5분
**다음 단계**: [3. 검증 및 수정](/docs/guide/process/iteration)에서 AI에게 수정을 요청하는 방법을 배우세요.
