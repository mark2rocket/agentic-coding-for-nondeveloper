## 📚 Step 1: 이해하기 (10분)

### 에이전틱 코딩 = AI가 대신 코딩

**전통적 방식:**
```
사람이 직접 코드 작성 → 몇 시간 소요
```

**에이전틱 방식:**
```
AI에게 요청 → 몇 분 만에 완성
```

### 핵심 3가지

1. **자율성**: AI가 스스로 판단
2. **반복성**: 에러 나도 자동 수정
3. **도구 사용**: 브라우저, 터미널 직접 조작

<Callout type="info" title="비유">
AI를 "진짜 개발자 동료"처럼 생각하세요. 요청하면 알아서 해줍니다!
</Callout>

## 🛠️ Step 2: 설치하기 (20분)

### 옵션 A: 안티그래비티 (완전 초보 추천)

**장점:** GUI, 쉬움, 시각적
**단점:** 느림, 정확도 낮음

**설치:**
1. Google Antigravity 검색
2. 공식 사이트 접속
3. 다운로드 버튼 클릭
4. 설치 파일 실행
5. 회원가입 및 로그인

**소요 시간:** 10분

---

### 옵션 B: 클로드코드 (중급 추천) ⭐

**장점:** 정확, 빠름, 강력
**단점:** 터미널 사용, 학습 곡선

#### Windows 사용자

**방법 1: GUI 설치 (추천)**
```powershell
# 1. 공식 사이트에서 다운로드
https://code.claude.com/download

# 2. ClaudeCodeInstaller.exe 실행
# 3. 설치 마법사 따라하기
```

**방법 2: winget (고급)**
```powershell
# PowerShell 관리자 권한으로 실행
winget install AnthropicPBC.ClaudeCode
```

#### macOS 사용자

**방법 1: GUI 설치 (추천)**
```bash
# 1. 공식 사이트에서 다운로드
https://code.claude.com/download

# 2. ClaudeCodeInstaller.dmg 실행
# 3. Applications 폴더로 드래그
```

**방법 2: Homebrew (고급)**
```bash
# 터미널에서
brew install claude-code
```

#### 설치 확인

```bash
# 터미널 열기 (Windows: PowerShell, Mac: Terminal)
claude --version

# 출력 예시:
# claude-code version 2.5.0
```

**소요 시간:** 15분

---

### API 키 설정 (클로드코드만)

1. **API 키 발급**
   ```
   https://console.anthropic.com/
   → API Keys 메뉴
   → Create Key
   ```

2. **API 키 입력**
   ```bash
   claude auth login
   # API 키 붙여넣기
   ```

3. **확인**
   ```bash
   claude auth status
   # ✅ Authenticated
   ```

## 🎨 Step 3: 첫 웹사이트 만들기 (20분)

### 프로젝트 생성

**안티그래비티:**
```
1. New Project 버튼 클릭
2. 프로젝트 이름 입력: "my-first-website"
3. Create 클릭
```

**클로드코드:**
```bash
# 1. 프로젝트 폴더 생성
mkdir my-first-website
cd my-first-website

# 2. Claude Code 실행
claude code
```

---

### 웹사이트 만들기

#### 안티그래비티 (GUI)

채팅창에 입력:
```
간단한 웹사이트 만들어줘.

내용:
- 제목: "내 첫 웹사이트"
- 부제목: "에이전틱 코딩으로 만들었어요!"
- 소개 문구: "안녕하세요. 저는 [이름]입니다."
- 버튼: "더 알아보기" (클릭하면 알림)

디자인:
- 배경: 그라데이션 (파랑 → 보라)
- 중앙 정렬
- 현대적이고 깔끔한 느낌
```

AI가 자동으로:
1. HTML 파일 생성
2. CSS 스타일 적용
3. JavaScript 이벤트 추가
4. 브라우저 자동 열어서 확인

**소요 시간:** 2분

---

#### 클로드코드 (CLI)

터미널에 입력:
```
간단한 웹사이트 만들어줘.

파일 구조:
- index.html (메인 페이지)
- style.css (스타일)
- script.js (인터랙션)

내용:
- 제목: "내 첫 웹사이트"
- 부제목: "에이전틱 코딩으로 만들었어요!"
- 소개 문구: "안녕하세요. 저는 [이름]입니다."
- 버튼: "더 알아보기" (클릭하면 알림 표시)

디자인:
- 배경: 그라데이션 (파랑 → 보라)
- 중앙 정렬
- 현대적이고 깔끔한 느낌
- 반응형 (모바일 대응)
```

AI가 자동으로:
1. `index.html` 생성
2. `style.css` 생성
3. `script.js` 생성
4. 파일 간 연결

**확인:**
```bash
# 간단한 HTTP 서버 실행
python -m http.server 8000

# 또는 (Python 없으면)
npx serve
```

브라우저에서 `http://localhost:8000` 접속

**소요 시간:** 5분

## 🚀 Step 4: 기능 추가하기 (10분)

### 추가 기능 1: 시계

**요청:**
```
현재 시간을 실시간으로 보여주는 시계 추가해줘.
위치는 페이지 상단 오른쪽.
```

AI가 자동으로:
- JavaScript로 현재 시간 가져오기
- 1초마다 업데이트
- 스타일 적용

**소요 시간:** 2분

---

### 추가 기능 2: 다크 모드

**요청:**
```
다크 모드 토글 버튼 추가해줘.
페이지 왼쪽 상단에 배치.
클릭하면 다크/라이트 모드 전환.
```

AI가 자동으로:
- 토글 버튼 생성
- 다크 모드 스타일 정의
- localStorage에 선택 저장

**소요 시간:** 3분

---

### 추가 기능 3: 애니메이션

**요청:**
```
페이지 로드할 때 제목이 위에서 부드럽게 내려오게 해줘.
버튼에 호버 효과도 추가해줘 (커지고 그림자 생김).
```

AI가 자동으로:
- CSS 애니메이션 추가
- transition 효과 적용

**소요 시간:** 2분
